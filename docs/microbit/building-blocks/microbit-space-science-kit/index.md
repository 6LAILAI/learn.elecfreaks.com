# micro:bit 太空科创套装

## 简介
---

太空科创套装是一款专为中小学生设计的STEAM教育套件，以哪吒多功能扩展盒V2为核心，配备行星系列智能马达和多种行星传感器，结合登月地图与丰富的积木结构件，让孩子们在动手构建和登月挑战中，轻松掌握航空科学，激发科学探索的热情。
