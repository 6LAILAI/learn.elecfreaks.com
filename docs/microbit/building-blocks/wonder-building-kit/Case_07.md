## 编写程序
### 添加专属扩展库

在MakeCode的代码抽屉中点击高级，查看更多代码选项，点击扩展。

![](./images/case-01-03.png)

在搜索框中搜索wukong，点击图片中wukong，添加扩展包。

![](./images/case-01-04.png)

在搜索框中搜索https://github.com/elecfreaks/pxt-sonarbit，添加超声波传感器的扩展包。
