1.	请使用 5V 2A 的充电适配器进行充电
2.	在酷霸智能车关闭的状态下，连接电源。
3.	充电状态下电量指示灯会显示当前电量，并闪烁
4.	当充满电时，电源指示灯全部点亮，请断开电源，完成充电。
