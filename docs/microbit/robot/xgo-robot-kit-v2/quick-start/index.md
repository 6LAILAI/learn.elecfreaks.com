# 教程内容介绍

XGO-lite V2 作为一种有趣的机器狗，其设计灵感来自于狗的四肢和步态，可以进行自主行走、跑步和动态平衡等功能，并且可以进行编程控制，更好的帮助您以及对仿生机器人感兴趣的创客学习编程知识以及机器人技术。

这个教程大纲共包含11个主题，每个主题都涉及到了不同的知识点和技能，以此帮助您了解和掌握仿生四足机器狗的基本运作原理和编程方法。
