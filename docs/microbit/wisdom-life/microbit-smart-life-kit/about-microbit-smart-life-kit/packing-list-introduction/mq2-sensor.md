# MQ2传感器

MQ2模拟气体传感器用于消费和工业市场的气体泄漏检测设备中，适用于检测LPG，异丁烷，丙烷，甲烷，酒精，氢，烟气，具有高灵敏度和快速响应时间灵敏度可通过电位器调节。
