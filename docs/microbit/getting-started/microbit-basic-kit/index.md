# 基础套件

## 简介

---
ELECFREAKS micro:bit basic kit是一款入门级的套件。我们精选了5款最常用的电子积木,他们能通过basic:bit方便地接入micro:bit主板。
这款套件能帮助学生们学习如何快速制作他们的micro:bit项目。这些电子积木可以被makecode默认的积木块直接驱动，无需加载额外的代码库，有利于学生从原理上理解电子积木的使用。
