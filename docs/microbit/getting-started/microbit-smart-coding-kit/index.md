# 手表套件

## 简介
---
micro:bit手表套件Pro(smart coding kit)是一套基于micro:bit纽扣电池扩展板Pro的可穿戴装备，仅需简单的装配步骤，就可DIY一款micro:bit手表。

micro:bit手表套件Pro基于一代手表套件进行了升级，

取消了亚克力后盖板，装配更加简单。

取消了3Pin端口，保持极简风格。

增加了蜂鸣器开关，功能再升级。
