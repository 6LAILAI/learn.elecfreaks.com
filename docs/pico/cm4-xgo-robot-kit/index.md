# XGO机械狗

## 简介

XGO-lite2是一款具有十五自由度桌面级Al机器狗，背部搭载机械臂和末端夹爪，内置树莓派CM4模组实现AI边缘计算应用，采用2.3KG.CM总线串口舵机作为关节，可实现全向移动、六维姿态控制、姿态稳定、多种运动步态和夹持任务，内部搭载IMU、关节位置传感器和电流传感器反馈自身姿态和关节转角与力矩，用于内部算法和二次开发。支持跨平台图形化，Python编程和ROS编程。
