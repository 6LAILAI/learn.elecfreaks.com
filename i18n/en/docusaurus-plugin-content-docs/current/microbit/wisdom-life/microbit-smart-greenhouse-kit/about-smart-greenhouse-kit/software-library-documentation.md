# Library Documentation

The IoT-environment-kit uses micro:bit as the main control board and Microsoft makecode platform as the programming platform. To facilitate users to program the iot-environment-kit, the ELECFREAKS team has developed the iot-environment-kit library file, and the iot-environment-kit contains 8 rainbow light ring components, so it needs to use the official neopixel library file as well. The following section describes in detail the role and parameter range of each programming command used in the kit:
