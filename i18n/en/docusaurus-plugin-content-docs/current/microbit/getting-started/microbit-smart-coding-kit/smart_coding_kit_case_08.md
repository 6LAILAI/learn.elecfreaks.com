# Case 08: Play the Music

## Purpose

 Use the Smart Coding Kit to play a music.

## Products Link

 1 x [ELECFREAKS micro:bit Smart Coding Watch Kit](https://www.elecfreaks.com/micro-bit-smart-coding-kit.html)
